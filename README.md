## Features

- **City Pop Aesthetic** — retro 80s color palette with hot pink, electric blue, and pastel accents
- **Smooth Animations** — Framer Motion page transitions + Lenis smooth scrolling
- **3D Scene** — Three.js with React Three Fiber rendering
- **Dark / Light Mode** — full theme switching support
- **Project Case Studies** — detailed write-ups with problem → process → result structure
- **Interactive Resume** — animated skill bars, exportable/printable
- **Music Player** — built-in ambient music widget
- **Responsive** — mobile-first design

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page — hero, skills, works, activities, contact |

| `/projects/[slug]` | Individual project case study |
| `/about` | Bio, background, and experience |
| `/resume` | Full interactive resume |
| `/activities` | Events and activities |

## Tech Stack

```
Framework    Next.js 16 (App Router)
Language     TypeScript 5.9
Styling      Tailwind CSS v4
Animation    Framer Motion 12
3D           Three.js + React Three Fiber
Scrolling    Lenis
Deployment   Vercel
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
